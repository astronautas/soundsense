# SoundSense
SoundSense - a SoundCloud player. Built with React and Rails as a web application assignment for Necolt.

Features:
* Streaming tracks from SoundCloud via built-in player.
* User profiles.
* Favorite tracks.

Technologies used:
* Rails 4
* Postgresql
* Bootstrap for Sass (Rails)
* Backbone.js
* Underscore.js
* jQuery
* React
